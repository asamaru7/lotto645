세금 부과율

5만원 이하
세금 0%
5만원 초과 ~ 3억원 이하
과세표준액의 기타소득세(20%) + 지방소득세(기타소득세의 10%) = 22% (각 원단위 절사)
3억원 초과
과세표준액의 기타소득세(30%) + 지방소득세(기타소득세의 10%) = 33% (각 원단위 절사)

예) 10억원에 당첨되었을 경우

과세표준액
10억원 - 1,000원(복권구입비) = 999,999,000원
3억원까지 22%
3억원 x 22% = 66,000,000원
3억원 초과 33%
699,999,000원 x 33% = 230,999,670원
총 세금
66,000,000원 + 230,999,670원 = 296,999,670원
실수령액
10억원 - 296,999,670원 = 703,000,330원
